# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Video Game Sales" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Данные представляют собой записи о продажах видеоигр по всему миру. Содержат информацию об играх (название, платформа, год выпуска, жанр, издатель) и продажах по регионам (Северная Америка, Европа, Япония, прочие, глобальные) в миллионах копий.<br>
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `data/` — датасет или скрипт загрузки  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения
